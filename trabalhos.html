Projeto Landing Page

O segundo projeto proposto se trata de uma Landing Page onde o meu tema foi o tema de uma Cafeteria Local. Nesse projeto enfrentei alguns desafios como a elaboração de menus dinâmicos e a implementação de um mapa no site, onde optei pelo uso de um script básico com JavaScript para a criação do menu e utilizei o recurso de incorporação de mapa que o Google Maps nos fornece.

Clique aqui para acessar o repositório do projeto! Clique aqui para acessar o site!

Projeto Banda

O terceiro projeto proposto trata-se da criação de um site dedicado a uma banda/artista musical de nosso gosto, onde eu escolhi o grupo Wu-Tang Clan como tema do projeto. O desafio desse projeto consistia em adicionar discografia da banda/grupo no projeto juntamente de algumas músicas de destaque, onde optei por incorporar as músicas do Spotify.
